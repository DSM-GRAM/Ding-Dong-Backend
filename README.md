# Ding-Dong - backend

## What is dong-dong
- ding-dong 은 '날짜피임법' 과 같은 기존 월경주기예측법과 함께 음성(목소리)을 분석하여 여성의 월경주기를 예측해주는 서비스입니다.
- ding-dong 은 메인 기능인 월경주기 예측을 중심으로 월경으로 인한 여성들의 생활 속 불편함을 해결해 주는 서비스 입니다.
